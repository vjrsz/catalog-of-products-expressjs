## Começando

### Pré-requisitos

* Node Js

### Instalação

1. Clonar o repositório
   ```sh
   git clone https://github.com/vjrsz/catalog-of-products-expressjs.git
   ```
2. Execute
   ```sh
   npm install
   npm run dev
   ```
